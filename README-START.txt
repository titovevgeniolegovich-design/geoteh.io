Готовый сайт собран в этой папке.

Что важно:
1. Откройте index.html
2. Если нужно подложить свои фоны - положите их сюда:
   assets\img\backgrounds\
3. Если нужно подложить инфографику PNG - положите её сюда:
   assets\img\infographics\
4. Если нужно заменить логотип - замените файл:
   assets\img\logo.svg
